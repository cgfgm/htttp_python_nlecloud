# NLECloud 图形化监控客户端

这是一个基于 `tkinter` 的 NLECloud 图形化客户端，支持：

- 用户名/密码登录并保存 `AccessToken` 到本地 `token.json`。
- 读取已保存 Token 后直接监控设备。
- 输入设备 ID 后点击“开始监控”轮询实时数据。
- 在表格中展示设备、ApiTag、当前值和记录时间。
- 将实时获取到的数值型数据绘制为折线趋势图。
- 通过温度滑块、阈值和传感器标识名自动发送执行器开关命令。
- 使用“手动打开 / 手动关闭”按钮直接控制指定传感器标识名。

## 运行环境

- Python 3.9+
- `tkinter`（通常随 Python 桌面版安装）

本客户端只使用 Python 标准库，不需要额外安装第三方包。

## 启动

```bash
python nlecloud_gui.py
```

## 使用步骤

1. 输入 NLECloud 用户名和密码，点击“登录”。登录成功后 Token 会保存到 `token.json`。
2. 后续启动程序可点击“读取已保存 Token”，无需重复输入密码。
3. 输入设备 ID，设置刷新间隔，点击“开始监控”。
4. 表格展示所有实时数据；数值型数据会自动追加到折线图中。
5. 在“温度联动控制”区域输入要控制的传感器标识名，并设置温度阈值。
6. 拖动温度滑块：当温度大于阈值时，程序会向该传感器标识名发送打开命令 `1`；否则发送关闭命令 `0`。
7. 如需绕过温度判断，可点击“手动打开”或“手动关闭”直接发送控制命令。
8. 点击“停止监控”停止轮询，点击“清空图表”清除表格和趋势图。

> `token.json` 包含访问凭证，请勿提交到版本控制或发送给他人。
