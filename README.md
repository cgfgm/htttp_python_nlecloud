# NLECloud 图形化监控客户端

这是一个基于 `tkinter` 的 NLECloud 图形化客户端，支持：

- 用户名/密码登录并保存 `AccessToken` 到本地 `token.json`。
- 读取已保存 Token 后直接监控设备。
- 输入设备 ID 后点击“开始监控”轮询实时数据。
- 在表格中展示设备、ApiTag、当前值和记录时间。
- 将实时获取到的数值型数据绘制为折线趋势图。
- 提供可增删的温度控制行，每行包含温度滑块、温度 ApiTag、阈值、传感器标识名、自动控制开关和手动打开/关闭按钮。
- 温度控制行可通过滑块手动设置温度，也可填写温度 ApiTag 后跟随实时数据自动更新。

## 运行环境

- Python 3.9+
- `tkinter`（通常随 Python 桌面版安装）

本客户端只使用 Python 标准库，不需要额外安装第三方包。

## 启动

```bash
python nlecloud_gui.py
```

## 使用步骤

1. 输入 NLECloud 用户名和密码，点击“登录”。登录成功后 Token 会保存到 `token.json`。
2. 后续启动程序可点击“读取已保存 Token”，无需重复输入密码。
3. 输入设备 ID，设置刷新间隔，点击“开始监控”。
4. 表格展示所有实时数据；数值型数据会自动追加到折线图中。
5. 在“温度自动控制”区域使用 `+` 新增一条温度控制行，使用 `-` 删除当前控制行（至少保留一条）。
6. 每条控制行可填写温度 `ApiTag`、阈值“值”和要控制的“传感器标识名”；当温度大于阈值时会向该传感器标识名发送 `1`（打开），否则发送 `0`（关闭）。
7. 温度既可通过滑块手动调整，也可在开始监控后由匹配的温度 `ApiTag` 实时数据自动更新。
8. 可勾选或取消“自动控制”，也可以点击“手动打开”或“手动关闭”立即控制对应传感器标识名。
9. 点击“停止监控”停止轮询，点击“清空图表”清除表格和趋势图。

> `token.json` 包含访问凭证，请勿提交到版本控制或发送给他人。
